76/*
77 Open a PDF document.
78
79 Open a PDF document by reading its cross reference table, so
80 MuPDF can locate PDF objects inside the file. Upon an broken
81 cross reference table or other parse errors MuPDF will restart
82 parsing the file from the beginning to try to rebuild a
83 (hopefully correct) cross reference table to allow further
84 processing of the file.

87 other PDF functions. Note that it wraps the context, so those
88 functions implicitly get access to the global state in
89 context.
