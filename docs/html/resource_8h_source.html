33 * Structures used for managing resource locations and avoiding multiple
34 * occurrences when resources are added to the document. The search for existing
35 * resources will be performed when we are first trying to add an item. Object
36 * refs are stored in a fz_hash_table structure using a hash of the md5 sum of
37 * the data, enabling rapid lookup.
38 */
